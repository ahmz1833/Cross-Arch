# Cross-Arch

A repo for referencing and installing toolchain of some useful architecture on linux x86_64 base architecure. (For running at Windows and MacOS you can use WSL or emulator or Docker - Instrcutions will be provided later)

## Planned Architectures

- MIPS (`mipsel` or also `mips32el`)
- IBM S390x (`s390x-z13`)
- ARM

## Will be added

- x86_64 architecture references, scripts, and samples
- Judge handling for each of architectures
